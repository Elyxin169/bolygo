## Verziókezelés
- helyi repo inicializálása
    > git init
- ellenőrzés
    > git status
- előkészítés
    > git add
- ellenőrzés
    > git status
- commitolás
    > git commit -m ""
- összes verzió
    > git log
- távoli repo létrehozása (github)
- összekapcsolás (helyi - távoli)
    > git remote add origin https://token@github.com/Elyxin169/repo-name.git
